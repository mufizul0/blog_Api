## GETTING STARTED
1. Clone the project
 
   ```
   git clone https://github.com/Ayine-nongre/Blog-API.git
   ```
2. Change to project directory

    ```
    cd Blog-API/
    ```
3. Set up enviroment variables in env file

   ```
   DB_USER, DB_PASSWD, PRIVATEKEY
   ```
4. Install packages used in project

   ```
   npm install
   ```
5. Start server

    ```
    nodemon app.js
    ```
